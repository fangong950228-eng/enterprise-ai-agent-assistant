# 财务报销制度

## 差旅报销
员工因公出差产生的交通、住宿、餐饮费用可以按规定报销。报销时需要提供发票、行程单、出差审批单。

## 打车报销
因公出行产生的打车费用可以报销。单次打车金额超过 200 元时，需要补充说明原因，并由直属主管审批。

## 餐费报销
因业务招待产生的餐费可以报销。报销时需要提供餐饮发票、消费明细和业务招待说明。

## 住宿报销
员工出差住宿需要选择符合公司标准的酒店。报销时需要提供住宿发票、酒店订单截图和出差审批记录。

## 办公采购报销
员工因工作需要采购办公用品、软件服务或小额设备时，需要提前获得直属主管确认。报销时需提供发票、采购明细和用途说明。

## 报销流程
员工需要在 OA 系统提交报销申请，上传发票和相关证明材料，经直属主管审批后提交财务审核。财务审核通过后进入付款流程。

## 发票要求
发票抬头、税号、金额、日期和项目内容必须清晰准确。发票内容与实际业务不一致时，财务有权退回申请。
