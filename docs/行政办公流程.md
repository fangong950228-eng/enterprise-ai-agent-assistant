# 行政办公流程

## 会议室预定
员工需要提前在 OA 系统中预定会议室，填写会议时间、会议主题、参会人数和设备需求。

## 办公用品申请
员工申请办公用品需要在行政系统提交申请，说明用品名称、数量和用途。常规用品由行政专员审批，大额用品需要部门负责人审批。

## 合同盖章
合同盖章需要提交合同终版、审批记录和用印申请。行政部门审核材料完整后安排盖章。

## 快递寄送
员工寄送公司文件或物品需要填写寄件信息、收件人信息和寄送原因。重要合同或证件类材料需要选择可追踪快递服务。

## 访客接待
外部访客到访前，接待员工需要提前在访客系统提交预约申请，填写访客姓名、公司、来访时间和来访事由。

## 固定资产领用
员工领用电脑、显示器、办公设备等固定资产时，需要在资产系统登记领用记录，并在离职或岗位变动时归还。
