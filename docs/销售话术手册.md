# 销售话术手册

## 客户关注成本
如果客户认为 AI 办公系统成本较高，可以从小范围试点、提升效率、减少重复劳动和长期 ROI 四个角度沟通。

示例话术：
“我们建议先从一个部门或一个高频场景开始试点，比如行政问答、销售跟进或日报生成。这样前期投入可控，也能通过实际数据评估节省的人力时间和流程效率提升。”

## 客户关注数据安全
如果客户担心数据安全，可以强调权限控制、私有化部署、日志审计和数据隔离。

示例话术：
“系统可以根据企业要求进行权限隔离，不同岗位只能访问授权范围内的知识库内容，同时保留使用日志，方便后续审计和追踪。”

## 客户不了解 AI Agent
可以将 AI Agent 解释为能够理解任务、调用工具、查询知识库并自动完成业务流程的智能助手。

示例话术：
“您可以把 AI Agent 理解成一个懂公司流程的数字员工，它不仅能回答问题，还能根据任务调用知识库、表单、系统接口，帮助员工完成重复性办公流程。”

## 制造业客户沟通角度
制造业客户通常关注降本增效、流程标准化和人员效率。建议优先推荐销售跟进、行政问答、制度查询和日报生成场景。

## 客户关注落地效果
如果客户担心 AI 项目落地效果不稳定，可以建议先做 PoC 验证，选取 20-50 个真实业务问题进行测试，并统计准确率、节省时间和人工反馈。
